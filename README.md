# shavon-qr-sub
## 概要
Shavon は、動画内に表示された QRコード を用いて 副音声を同期再生するためのシステムです。 動画には音声そのものではなく 「何を・いつ・どのように読み上げるか」という情報を埋め込み、 音声生成と再生は視聴側の PC で行います。 これにより、プラットフォームや動画形式に依存しない 柔軟な副音声再生を可能にします。想定用途 •解説・コメンタリー付き動画 •実験的・技術的な映像作品 •複数言語・複数音声の切り替え •プラットフォーム非依存の副音声提供 主な特徴 •動画プラットフォーム非依存 •音声データを動画に含めない設計 •再生位置・再生速度にリアルタイム追従 •早送り・巻き戻し・シーク操作に対応 •VOICEVOX を用いたローカル音声生成 •ネットワーク通信量はほぼ不要

## システム構成
Shavon は主に 3 種類の QR 情報を扱います。
セリフQR（Speech QR）
副音声の内容を定義します。
含まれる情報例：
•読み上げテキスト
•話者（VOICEVOX キャラクター）
•話速・抑揚などの音声パラメータ
•再生予定位置（論理時刻またはID）
読み取り後、音声は事前に生成されバッファされます。
タイミングQR（Timing QR）
現在の再生位置を示す基準情報です。
•動画内の論理時刻
•フレーム番号
•セリフ進行インデックス など
Shavon は動画プレイヤーの内部状態を参照せず、
この QR を 唯一の同期基準 として使用します。
再生制御
タイミングQRの変化量から再生速度を推定し、
副音声を以下のように制御します。
•再生位置の即時シーク
•再生速度への追従
•早送り時の高速再生
早送り・倍速再生への対応
生成済み音声は SoundTouch による
Time‑Stretch 処理を用いて再生されます。
•再生速度のみを変更
•ピッチ（声の高さ）は維持
これにより、動画の倍速再生や早送り中でも
聞き取り可能な形で副音声が追従します。
技術的特性
•音声生成：VOICEVOX（ローカル実行）
•再生制御：Time‑Stretch（SoundTouch）
•映像解析：PC内部キャプチャ＋QRデコード
•同期方式：動画内タイミングQRによる同期
## 名称説明
•Shavon：Shabon（泡）＋ V（Voice / VOICEVOX）
•BubbleVox：英語圏向け副題
•QR副音声再生システム：機能説明名称
声を一度シャボン玉に包み 受け取り手に届ける事をイメージしています。受け手側はその泡の中から必要な情報を自由に割って取り出し使用できる事を目指しています。
