# shavon-qr-sub
概要 Shavon は、動画内に表示された QRコード を用いて 副音声を同期再生するためのシステムです。 動画には音声そのものではなく 「何を・いつ・どのように読み上げるか」という情報を埋め込み、 音声生成と再生は視聴側の PC で行います。 これにより、プラットフォームや動画形式に依存しない 柔軟な副音声再生を可能にします。想定用途 •解説・コメンタリー付き動画 •実験的・技術的な映像作品 •複数言語・複数音声の切り替え •プラットフォーム非依存の副音声提供 主な特徴 •動画プラットフォーム非依存 •音声データを動画に含めない設計 •再生位置・再生速度にリアルタイム追従 •早送り・巻き戻し・シーク操作に対応 •VOICEVOX を用いたローカル音声生成 •ネットワーク通信量はほぼ不要
